  • Go 在開發時，會立刻針對型別做檢查

slices 資訊 (重要觀念)

如果想要得知目前 slice 長度跟複製的陣列來源長度，可以透過 len() 和 cap() 兩個方法取得

  • len() : slice 出來的陣列長度
  • cap(): slice 來源的陣列長度

預設值

而一個 slices 沒有設定初始值時，預設值就是 nil

make 是 go 內建的方法之一，可以透過 make 來建立一個動態長度的陣列, slices. 語法是
